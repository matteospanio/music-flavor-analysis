music-flavor-analysis

Analisi statistica per studio “Musica e Gusto”

Formalizzazione

Ogni brano ha un vettore target (ground truth) associato:

\[ \mathbf{t}_i = (salty, sweet, sour, bitter, spicy) \]

(metadata del dataset / query target)

  • Ogni partecipante ascolta 10 brani campionati casualmente da un pool di 20 brani
  • Per ogni brano fornisce un vettore percettivo Likert (1-7): \[ \mathbf{p}_{ij} = (salty, sweet, sour, bitter, spicy) \]

Dove i = brano, j = partecipante

Ipotesi

\(H_0\): Il vettore percettivo non e’ correlato al vettore target (associazione casuale)

\(H_1\): Esiste una relazione sistematica tra metadate del brano e percezione gustativa

Normalizzazione (z-score)

Normalizzare perché le scale di valutazione possano essere confrontate (le scale likert sono diverse dai valori target), per questo abbiamo bisogno di fare \(z\)-score per ciascuna dimensione del vettore percettivo: \[ \mathbf{p}_{ij}^{norm} = \frac{\mathbf{p}_{ij} - \mu_j}{\sigma_j} \] dove \(\mu_j\) e \(\sigma_j\) sono la media e la deviazione standard delle valutazioni per la dimensione \(j\). Ci aspettiamo che la varianza spuria venga ridotta in questo modo.

  • Normalizzazione dei vettori percettivi:
d_long <- d_long %>%
  group_by(gusto) %>%
  mutate(rating_z = scale(rating)) %>%
  ungroup()

summary_norm <- d_long %>%
  group_by(gusto) %>%
  summarise(
    Mean = mean(rating_z),
    SD = sd(rating_z),
    .groups = "drop"
  )

Statistica descrittiva

Per dimensione gustativa

Calcolo della media e della deviazione standard delle valutazioni percettive per ogni dimensione gustativa.

  • media

    \[ \mu_j = \frac{1}{M} \sum_{i=1}^{20} \sum_{k=1}^{N_i} p_{ikj} \]

  • deviazione standard

    \[ \sigma_j = \sqrt{\frac{1}{M} \sum_{i=1}^{20} \sum_{k=1}^{N_i} (p_{ikj} - \mu_j)^2} \]

dove \(M = \sum_{i=1}^{20} N_i\) è il numero totale di valutazioni.

descrittive_gusti <- d_long %>%

    Deviazione_Standard = sd(rating),

L’ispezione delle distribuzioni delle valutazioni per ciascuna dimensione gustativa, mediante boxplot e istogrammi, è stata utilizzata per verificare l’eventuale presenza di ceiling e floor effects.

Per le dimensioni acido, amaro, dolce e piccante, i boxplot mostrano mediane intorno al valore 2, con quartili approssimativamente compresi tra 1 e 4 e whiskers che si estendono fino ai valori più alti della scala (7). Questo indica una leggera concentrazione delle risposte verso valori bassi, ma anche la presenza di valutazioni intermedie e alte. Non si osserva quindi un accumulo marcato esclusivamente sul valore minimo della scala.

Per la dimensione salato, la mediana risulta intorno al valore 3 e la distribuzione appare maggiormente centrata, con ampia dispersione lungo l’intera scala Likert. Non emergono concentrazioni evidenti agli estremi.

Gli istogrammi confermano questi pattern: per tutte le dimensioni è presente un picco di frequenza sul valore 1, ma sono ben rappresentati anche i valori compresi tra 2 e 6, con code che raggiungono il valore massimo della scala.

Nel complesso, le distribuzioni coprono l’intero range 1–7, indicando che la scala Likert è stata utilizzata in modo adeguato dai partecipanti.

Test principale di non-causalità (CORE CLAIM)

Permutation test sulla distanza vettoriale

  • permutation test su distanza target-perceived:

    • Per ogni brano i, calcolare la distanza tra vettore target e vettore percettivo medio:

      \[ d_i = \| \mathbf{t}_i - \bar{\mathbf{p}}_i \| \]

    • Calcolare la distanza media osservata:

      \[ \bar{d}_{obs} = \frac{1}{20} \sum_{i=1}^{20} d_i \]

    • Creare una distribuzione nulla delle distanze medie tramite permutazione:

      • Per k = 1 a 10.000:

        • Permutare casualmente le associazioni tra brani e valutazioni percettive.

        • Per ogni brano i, calcolare la distanza tra vettore target e vettore percettivo medio permutato:

          \[ d_i^{perm} = \| \mathbf{t}_i - \bar{\mathbf{p}}_i^{perm} \| \]

        • Calcolare la distanza media permutata: \[ \bar{d}_{perm}^{(k)} = \frac{1}{20} \sum_{i=1}^{20} d_i^{perm} \]

      • Ottenere la distribuzione nulla delle distanze medie permutate:

        \[ D_{null} = \{ \bar{d}_{perm}^{(1)}, \bar{d}_{perm}^{(2)}, \ldots, \bar{d}_{perm}^{(10000)} \} \]

    • Calcolare il p-value come la proporzione di distanze medie permutate che sono minori o uguali alla distanza media osservata:

      \[ p = \frac{1}{10000} \sum_{k=1}^{10000} I(\bar{d}_{perm}^{(k)} \leq \bar{d}_{obs}) \]

      dove I è la funzione indicatrice.

set.seed(123)      # riproducibilità
n_perm <- 10000

d_perm_means <- numeric(n_perm)
d_obs <- mean(dist_per_brano$d)   # distanza media osservata

for (k in 1:n_perm) {
  merged_perm <- merged

  idx <- sample(1:nrow(merged_perm))

  merged_perm[, c("acido_perc", "amaro_perc", "dolce_perc", "piccante_perc", "salato_perc")] <-
    merged_perm[idx, c("acido_perc", "amaro_perc", "dolce_perc", "piccante_perc", "salato_perc")]

  d_k <- merged_perm %>%
    rowwise() %>%
    mutate(
      d = sqrt(
        (acido_tgt - acido_perc)^2 +
          (amaro_tgt - amaro_perc)^2 +
          (dolce_tgt - dolce_perc)^2 +
          (piccante_tgt - piccante_perc)^2 +
          (salato_tgt - salato_perc)^2
      )
    ) %>%
    ungroup()

  d_perm_means[k] <- mean(d_k$d)
}

Dnull <- d_perm_means
p_value <- mean(Dnull <= d_obs)
risultati_perm <- tibble::tibble(
  `Permutazioni` = n_perm,
  `Distanza media osservata` = d_obs,
  `Media nulla` = mean(Dnull),
  `SD nulla` = sd(Dnull),
  `p-value` = p_value
)
